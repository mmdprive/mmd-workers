# admin-console-nextjs-skeleton

Next.js + Tailwind skeleton for MMD Privé Admin Console / Deal Control Panel.

## Included pages
- /admin
- /admin/console
- /admin/console/dashboard
- /admin/console/deals
- /admin/console/deals/[deal_id]
- /admin/console/models
- /admin/console/clients
- /admin/console/payments

## Run
```bash
npm install
npm run dev
```

## Notes
- Mock data included in `lib/mock.ts`
- API helper stubs included in `lib/admin-api.ts`
- Tailwind theme tuned for black x gold MMD styling
